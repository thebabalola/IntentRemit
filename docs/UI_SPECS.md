# UI Specification Guide
\n## Onboarding Modal Definitions\n- Onboarding Modal: General popup for first-time users.\n- Product Tour: Step-by-step feature guidance.\n- Welcome Modal: Brief intro greeting.\n- Guided Tour: Active UI walkthrough.
\n## Loading States\n- Skeleton Loader: Fake UI preview while loading.\n- Spinner/Loader: Rotating indicator.\n- Shimmer Effect: Animated skeleton.
\n## Empty States\n- No transactions yet\n- No items in cart\n- Guides users on next actions.
\n## Error & Success States\n- Error: Network error, failed to load data.\n- Success: Toast notifications, alerts, confirmations.
